# M-07 钉钉集成模块

## 概述
M-07 钉钉集成模块负责与钉钉平台的集成，包括：
1. 通过钉钉机器人发送通知
2. 钉钉单点登录(SSO)认证
3. 钉钉组织架构同步
4. 支持发送带链接的通知消息

## 数据模型
涉及的核心数据模型：
- User 表：增加 `dingtalkUserId` 字段用于存储钉钉用户ID，实现SSO登录绑定

## API 接口

### 钉钉SSO相关
- `GET /api/auth/dingtalk/callback` - 钉钉SSO登录回调接口
  - Query参数：`?code={临时授权码}&state={状态参数}`
  - 响应：重定向到前端登录成功页面，携带token

### 通知发送相关
- `pushPendingDingtalkNotifs` - 批量推送钉钉通知（工具函数）
  - 支持发送文本消息和链接消息
  - 链接消息包含可点击的URL，用户可跳转到系统

## 业务逻辑
1. 用户点击钉钉登录按钮 → 跳转到钉钉授权页面 → 用户授权后钉钉回调到 `/api/auth/dingtalk/callback` → 系统验证后生成JWT并重定向到前端登录成功页面
2. 系统发送通知时，可通过 `linkUrl` 参数发送带链接的消息，用户点击后可跳转到系统相关页面
3. 首次SSO登录时，系统会根据钉钉userid查找或匹配本地用户并建立绑定关系

## 前端组件
- `DingtalkSsoButton` - 钉钉SSO登录按钮组件
- `DingtalkNotificationDemo` - 钉钉通知发送演示组件
